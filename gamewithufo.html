UFO Part Time

👽

is a game in which you play as a character named Jobski, who is a UFO. In the game, you can choose several jobs focused on taking and grasping objects and then moving them to another place. There are, for example, jobs involving the creation of buildings. You create buildings by transferring blocks with a crane that UFO Jobski has .Buildings are then tested with various stress tests for five seconds.The goal of the game is to complete the work.UFO Part Time is a game released for Android and IOS, but a port for Nintendo Switch was added in 2020.The game is created in Unity and Cocos2d-x, for example, the game Hungry Shark Arena is created in Cocos creator.
