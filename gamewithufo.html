Part Time UFO

👽

the main character of Part time UFO is Jobski, Jobski is a UFO or more precisely Flying Saucer. Player can choose several jobs focused on taking and grasping objects and then moving them to another places in Part Time UFO. There are, for example, jobs involving the creation of buildings. Player can create buildings by transferring blocks with a crane that UFO/Flying Saucer Jobski has. Buildings are then tested with various stress tests for five seconds. The goal of the game is to complete the work. Part Time UFO is a game released for Android and IOS, but a port for Nintendo Switch was added in 2020.The game is created in Unity and Cocos framework, 2D games like a Hungry Shark Arena is also created in Cocos framework.
